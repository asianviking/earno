# Agent integration

earno is agent-native via `incur`.

## MCP server

Register earno as an MCP server (for Claude Code, Cursor, etc.):

```sh
earno mcp add
```

## Skill files

Sync skill files to your agent:

```sh
earno skills add
```

## LLM-readable manifest

Print the command manifest:

```sh
earno --llms
```
